# HTML Review – 005: Multimedia

## 📖 About

This project is the fifth exercise in my **HTML Review Series**.

The purpose of this project is to explore how multimedia is handled in HTML using semantic elements and modern best practices. Instead of only demonstrating the syntax, this exercise explains when and why different attributes should be used to improve accessibility, performance, responsiveness, and user experience.

The project covers images, videos, audio, responsive media, subtitles, and several optimization techniques commonly used in modern web development.

---

## 📚 Topics Covered

### 🖼 Images

* `<img>`
* `<picture>`
* `src`
* `alt`
* `width` & `height`
* `loading="lazy"`
* `decoding="async"`
* `srcset`
* `sizes`
* `<source>` for responsive images and modern image formats

### 🎥 Videos

* `<video>`
* `<source>`
* `controls`
* `poster`
* `muted`
* `loop`
* `autoplay`
* `preload`
* Multiple video formats

### 💬 Subtitles & Captions

* `<track>`
* `kind`
* `srclang`
* `label`
* `default`
* WebVTT (`.vtt`) subtitle files

### 🎵 Audio

* `<audio>`
* Playback controls
* `preload`
* Common playback attributes
