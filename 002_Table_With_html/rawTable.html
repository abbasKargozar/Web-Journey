How to create A table in html

There are some essential tags below:

<table> </table>
This is base for every table
<th> </th>
We can create table header by this
<tr> </tr>
It creates a new row for table
<td> ></td>
Use this inside of <tr> </tr> to create diffrent column
<caption>...</caption>
write your table caption between this tag

For Example:

This is a Table without "border attribute"

Forename Surname City Age job Salary
Charlie Ashford London 23 Chef $7000
Annie Ravenwood Manchester 20 weitress $4000

This is a table with "border" attribute

Forename Surname City Age job Salary
Charlie Ashford London 23 Chef $7000
Annie Ravenwood Manchester 20 weitress $4000

*border attribute: <table border = "1"> ... </table>*


A little bit complex table

Do you want to draw a table like the below one? Don't worry it's really simple and all things you need are colspan="number" & rowspan="number" inside of th or td tag like:

<th colspan = "2"> </th> or <td rowspan = "2"> </td>

or

<td rowspan = "2"> </td> 
        -----------------------------------------------------------------
        |                        Weekly Schedule                        |
        -----------------------------------------------------------------
        |    Time    |    Mon    |    Tue    |    Wed    |     Thu    |

        -----------------------------------------------------------------

        -----------------------------------------------------------------

        -------------- Chemistry -------------          History         -
        |   12:00    |           |    Art    |                          |
