# HTML Review – 002: Tables

## 📖 About

This project is the second exercise in my **HTML Review Series**.

The goal of this review is **not** to build a beautiful webpage, but to strengthen my understanding of HTML fundamentals before moving on to CSS, JavaScript DOM manipulation, and Web Application Security.

For this reason, styling is intentionally kept to an absolute minimum.

---

## 🎯 Objectives

In this exercise I practiced:

- Creating HTML tables
- Table headers (`<th>`)
- Table rows (`<tr>`)
- Table cells (`<td>`)
- Table captions (`<caption>`)
- `colspan`
- `rowspan`

### 2. Table with Border

### 3. Complex Table

A more advanced table demonstrating:

- Merged columns (`colspan`)
- Merged rows (`rowspan`)

---

## 💡 Learning Notes

During this exercise I learned that:

- Every table starts with the `<table>` element.
- Every row is created using `<tr>`.
- `<th>` is used for table headers.
- `<td>` stores table data.
- `rowspan` merges cells vertically.
- `colspan` merges cells horizontally.
- Cells covered by `rowspan` or `colspan` should **not** be created again.
